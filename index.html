Langkah Belajar HTML Saya

  1. Membuat file HTML Pertama (Hello World)
  2. Memahami struktur DOCTYPE, head, dan body
  3. Belajar element eks: heading, paragraf, strong
  4. Membuat link, gambar, dan list
  5. Membangun halaman profil semantik lengkap
